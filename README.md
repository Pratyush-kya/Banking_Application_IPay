# 💳 iPay - Banking Application

iPay is a modern Android banking application developed using **Kotlin** and **Android Studio**. The application demonstrates secure user authentication, account management, transaction tracking, financial analytics, market information, currency conversion, and local data storage.

The project follows a modular architecture and integrates Firebase services, local database management, networking, and modern Android development practices.

---

## ✨ Features

- 🔐 User Authentication
  - Login
  - Signup
  - Firebase Authentication

- 👤 User Profile Management

- 📊 Dashboard
  - Account Overview
  - Financial Summary

- 💸 Transaction Management
  - Transaction Details
  - Transaction History

- 💱 Currency Converter

- 📈 Market Information

- 💬 Chat Module

- 📉 Financial Charts

- 🔒 Biometric Authentication

- ☁️ Cloud Database Integration

- 💾 Offline Data Storage using Room Database

# 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Pratyush-kya/Banking_Application_IPay.git
```

### Open Project

Open the project in **Android Studio**.

### Configure Firebase

1. Create a Firebase Project.
2. Download the `google-services.json` file.
3. Place it inside:

```
app/
```

### Sync Gradle

Allow Gradle to download all dependencies.

### Run

Run the application on an emulator or Android device (API 26+).
